# Why the first version should be embarrassingly small

This week’s Telegram assistant taught me a simple rule: make v1 embarrassingly small. If the agent does one thing well and you can test it easily, you can build confidence quickly and iterate. My LinkedIn PR agency is nearly there because the workflow it automates is narrow and repeatable; the dance-studio directory still needs more data work because the task is broader.
